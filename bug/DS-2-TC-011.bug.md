## Summary

Playwright **TC-011** in `tests/pw-ds2U-edit_program.spec.ts` fails: renaming program A to match
program B's name (uppercased) is accepted with no visible feedback. Case-insensitive uniqueness
is not enforced on edit/rename.

## Steps to reproduce

1. Open **Edit** on program A
2. Set **Program Name** to B's name in UPPERCASE
3. Click **Save**

## Expected result

- Save rejected (modal stays open) AND visible feedback (alert/toast/inline) matching
  `/duplicate|already exists|must be unique|name taken/i` within 8s

## Actual result

- No alert/toast/inline duplicate error appears within 8s (element not found)
