## Summary

Playwright **TC-008** in `tests/ds4-delete-program.spec.ts` fails: double-clicking the Delete
button for a program opens **two** native confirmation dialogs instead of exactly one.
`dialogCount` reaches 2; assertion `expect.poll(() => dialogCount).toBe(1)` fails.

## Steps to reproduce

1. Sign in as **Admin** at https://test.didaxis.studio
2. Go to **Programs**
3. Create a program (e.g. `Double Delete <timestamp>`)
4. **Double-click** the Delete button for that program

## Expected result

- Exactly **one** confirmation dialog is shown
- Program remains in the list when dialogs are dismissed without confirming

## Actual result

- **Two** confirmation dialogs fire (`dialogCount = 2`)
- Playwright assertion at `tests/ds4-delete-program.spec.ts:118` fails
